# Blinkit-Sales-Dashboard
This repository contains an interactive Power BI analytics dashboard designed to analyze and visualize key business metrics for Blinkit, 
focusing on sales performance, outlet characteristics, product types, and customer ratings.

The Blinkit Power BI dashboard gives a comprehensive picture of:

Sales performance across outlet locations, sizes, and types

Item-level metrics such as average sales, rating, and visibility

Category-wise product performance

Trends in outlet establishment years

Impact of fat content on sales

Comparison of item types and their contribution to revenue

The dashboard is fully interactive, making it easier to identify patterns, explore trends, and derive actionable insights.
